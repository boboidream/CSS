# 盒子模型

- box-shadow
```css
/* inset | offset-x | offset-y | spread-radius | color */
box-shadow: inset 0 0 1em gold;

/* offset-x | offset-y | blur-radius | spread-radius | color */
box-shadow: 2px 2px 2px 1px rgba(0, 0, 0, 0.2);
```

- 强制不换行
```css
div{
white-space:nowrap;
}
```

- 如何保持固定比例
padding-top 的百分比是相对于宽度的，所以可以用来做比例布局
